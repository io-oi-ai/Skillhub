A collection of 15 skills designed to polish interfaces built by AI agents. Works with Claude Code, Cursor, OpenCode, and other platforms.

> **Source:** [ibelick/ui-skills](https://github.com/ibelick/ui-skills) — install the full collection with `npx skills add ibelick/ui-skills`

## All 15 Skills

| Skill | Description |
|-------|-------------|
| **baseline-ui** | Validates animation durations, enforces typography scale, checks component accessibility, and prevents layout anti-patterns in Tailwind CSS projects. |
| **fixing-accessibility** | Audits and fixes HTML accessibility issues: ARIA labels, keyboard navigation, focus management, color contrast, and form errors. |
| **fixing-metadata** | Audits and fixes HTML metadata: page titles, meta descriptions, canonical URLs, Open Graph tags, Twitter cards, and favicons. |
| **fixing-motion-performance** | Audits and fixes animation performance issues: layout thrashing, compositor properties, scroll-linked motion, and blur effects. |
| **12-principles-of-animation** | Applies Disney's 12 animation principles to web interfaces to enhance motion quality. |
| **canvas-design** | Creates original visual designs using design philosophy focused on form, space, and color. |
| **design-lab** | Interactive design exploration workflow: conducts interviews, generates variants, and refines UI designs. |
| **frontend-design** | Creates distinctive, production-grade frontend interfaces with high design quality. |
| **interaction-design** | Designs microinteractions, motion design, transitions, and user feedback patterns. |
| **interface-design** | Specialized for dashboards, admin panels, and SaaS applications. |
| **swiftui-ui-patterns** | Best practices and example-driven guidance for building SwiftUI views and components. |
| **tailwind-css-patterns** | Expert guidance for building responsive interfaces with Tailwind CSS utility-first patterns. |
| **ui-ux-pro-max** | 50+ styles, 97 palettes, and 9 technology stacks for professional interface design. |
| **wcag-audit-patterns** | Conducts WCAG 2.2 accessibility audits with automated testing and remediation guidance. |
| **web-design-guidelines** | Reviews UI code for Web Interface Guidelines compliance against Vercel's best practices. |

## Usage

```bash
# Install all skills
npx skills add ibelick/ui-skills

# Add individual skill
npx ui-skills add [skill-name]

# Example: review a directory
/baseline-ui review src/
```

## Skill Categories

- **Fixing / Auditing**: accessibility, metadata, motion performance, WCAG
- **Design**: canvas, frontend, interaction, interface, design-lab
- **Pattern Guidance**: Tailwind CSS, SwiftUI, animation principles, web design guidelines

# 中文版

# UI Skills 界面打磨工具集

## 技能概览

15 个独立技能，专门用于打磨 AI 生成的界面。涵盖设计一致性、无障碍、元数据、动画性能、交互设计、Tailwind/SwiftUI 模式等方面。

## 技能分类

- **审计修复类**：accessibility（无障碍）、metadata（SEO 元数据）、motion-performance（动画性能）、WCAG 审计
- **设计创作类**：canvas-design、frontend-design、interaction-design、interface-design、design-lab
- **模式指导类**：Tailwind CSS 模式、SwiftUI 模式、迪士尼 12 动画原则、Web 设计规范

## 使用建议

按需安装单个技能，或一次安装全部 15 个。适合在开发完成后对界面做系统性打磨和质量提升。
