A collection of skill files that make AI coding tools generate modern, premium frontend interfaces instead of generic ones.

> **Source:** [Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill) — copy the `SKILL.md` from each skill folder into your project

## Skills Included

### 1. taste-skill (Core)

The foundational skill covering layout, typography, colors, spacing, motion, and visual quality. Includes three configurable settings on a 1–10 scale:

| Setting | Low (1–3) | High (8–10) |
|---------|-----------|-------------|
| **DESIGN_VARIANCE** | Standard grids | Asymmetric, modern layouts |
| **MOTION_INTENSITY** | Minimal transitions | Spring physics, magnetic effects |
| **VISUAL_DENSITY** | Luxury spacing | Compact dashboards |

### 2. redesign-skill

Upgrades existing projects by auditing the current design and prioritizing fixes, rather than rebuilding from scratch.

### 3. soft-skill

Makes things look and feel expensive. Covers premium fonts, generous whitespace, layered card designs, smooth spring animations, and floating navigation. Actively avoids generic template patterns.

### 4. output-skill

Forces AI to write complete code. Eliminates placeholder comments, incomplete code blocks, and unfinished outputs — ensures full implementations every time.

### 5. minimalist-skill

Creates clean, editorial-style interfaces inspired by Notion and Linear. Uses warm monochrome palettes, serif/sans-serif contrast, massive whitespace, flat bento grids with hairline borders, and muted pastels. Blocks gradients and heavy shadows.

## Installation

No package installation required — just copy the `SKILL.md` file from the desired skill folder into your project root, then reference it in your AI tool (e.g., `@SKILL.md` in Cursor).

```bash
# Clone the repo
git clone https://github.com/Leonxlnx/taste-skill.git

# Copy the skill you want
cp taste-skill/taste-skill/SKILL.md ./SKILL.md
```

# Taste Skill 高品质前端设计技能

## 技能概览

5 个技能文件，让 AI 编码工具生成有质感的现代前端界面，而非千篇一律的模板风格。

## 包含技能

- **taste-skill**（核心）：可配置的设计方差、动效强度、视觉密度
- **redesign-skill**：审计并渐进升级现有项目设计
- **soft-skill**：打造"昂贵感" — 高级字体、宽阔留白、弹簧动画、浮动导航
- **output-skill**：强制 AI 输出完整代码，杜绝占位符和未完成片段
- **minimalist-skill**：编辑风格极简界面，灵感来自 Notion 和 Linear

## 使用建议

将对应的 `SKILL.md` 复制到项目根目录，AI 工具会自动遵循其中的设计规范。适合对审美有高要求的前端项目。
