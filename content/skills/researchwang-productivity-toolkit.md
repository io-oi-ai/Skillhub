## Overview

Five productivity skills that turn your OpenClaw agent from a reactive chatbot into a proactive assistant with memory, self-improvement, and automation capabilities.

## Skills

### 1. Self-Improving Agent

Your agent automatically reflects on its performance after each session and iteratively improves its workflows.

```markdown
# Install
帮我安装这个skill:
https://github.com/peterskoett/self-improving-agent/tree/master

# Configure daily auto-run
每天自动运行 self-improving-agent, 完成每日的记录和更新
```

### 2. Memory Setup

Gives your agent persistent long-term memory across sessions — it remembers past conversations, preferences, and project context.

```markdown
# Install
帮我安装这个skill:
https://clawhub.ai/jrbobbyhansen-pixel/memory-setup

# Test
帮我把记忆能力配置起来
你记得我们上次聊的内容吗？帮我回想起来
```

### 3. Summarize

Quickly distill long articles, web pages, or documents into concise summaries with key takeaways.

```markdown
# Install
帮我安装这个skill:
https://clawhub.ai/steipete/summarize

# Test
使用 Summarize 总结 https://example.com/long-article
```

### 4. Auto-Updater

Keeps all installed skills at their latest versions automatically — no manual checking required.

```markdown
# Install
帮我安装这个skill:
https://clawhub.ai/maximeprades/auto-updater

# Configure daily updates
帮我每天完成技能的自动更新 auto-updater
```

### 5. Proactive Agent

Instead of waiting for your commands, the agent monitors your schedule and proactively reminds you of tasks, deadlines, and follow-ups.

```markdown
# Install
帮我安装这个skill:
https://clawhub.ai/halthelobster/proactive-agent

# Configure
设置每天早上 9 点自动检查日程并提醒我待办事项
```

## Recommended Setup Order

1. **Memory Setup** — enable persistence first so nothing is lost
2. **Auto-Updater** — keep everything current automatically
3. **Summarize** — useful from day one for research
4. **Self-Improving Agent** — let the agent optimize itself over time
5. **Proactive Agent** — once comfortable, enable autonomous scheduling

# 中文版

# Agent 生产力工具箱

## 技能概览

该工具箱提供 5 类能力：自我改进、长期记忆、长文总结、自动更新、主动提醒，帮助 Agent 从被动问答升级为主动协作助手。

## 核心能力

- Self-Improving Agent：会后自我复盘与流程优化
- Memory Setup：跨会话长期记忆
- Summarize：长文/网页快速摘要
- Auto-Updater：技能自动升级
- Proactive Agent：主动日程提醒与跟进

## 推荐启用顺序

1. 先启用 Memory，保证上下文可持续
2. 开启 Auto-Updater，保持技能常新
3. 引入 Summarize，提升信息处理效率
4. 配置 Self-Improving，持续优化工作流
5. 最后启用 Proactive，进入主动协作模式

## 使用建议

先从“提醒 + 总结”两个低风险场景开始，再逐步放开自动化权限。
