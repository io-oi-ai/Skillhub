# SkillHub Homepage Redesign Design

## Reference
Fig Mint product page - light cream theme, serif titles, left-text-right-visual layout, feature grid, minimal CTA.

## Decisions
- Scope: entire homepage (Header, Hero, SkillGrid, Footer)
- Theme: light (cream/off-white), replacing current dark theme globally
- Typography: serif for titles (Georgia, Noto Serif SC), sans-serif for body
- Hero right side: pure CSS/SVG abstract decorative graphic
- Feature grid: platform features (free, open-source, multi-platform, one-click install)
- Approach: Classic Fig Mint style (Option A)

## Color System
- Background: `#f5f5f0` (warm off-white)
- Card background: `#ffffff`
- Border: `#e5e5e0`
- Primary/CTA: `#1a1a1a` (dark black)
- Primary text: `#1a1a1a`
- Secondary text: `#6b6b6b`
- Muted text: `#9a9a9a`

## Layout (top to bottom)

### Header
- Light background, no border
- Left: SkillHub logo (serif font)
- Right: nav links + dark submit button

### Hero (left-right split)
- Left (~50%): large serif title, description paragraph, dark CTA button, 2x2 feature grid
- Right (~50%): CSS/SVG abstract decorative graphic representing AI workflow

### Skill List
- Search bar + filter bar (light theme adapted)
- White cards with light gray borders, dark text
- Role/scene tags adapted for light theme

### Footer
- Minimal, light background, muted text
